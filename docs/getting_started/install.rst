.. _install:

installation
============

The SWAN source code is written in Fortran90 and is available at the `Gitlab repository <https://gitlab.tudelft.nl/citg/wavemodels/swan>`_.
You can either install the pre-compiled SWAN package directly on your computer or configure, build and install SWAN using a Fortran90 compiler.
Alternatively, you can run SWAN immediately using a docker container. Especially users new to SWAN might find this very helpful.
