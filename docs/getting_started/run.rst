run instructions
================

SWAN can be run on various architectures, including laptops, HPC machines and cloud-based clusters,
Depending on your choice of :ref:`installed SWAN <install>`, you can run the simulations locally on an OS platform (Windows, Linux or Mac)
via your own built SWAN or via Docker. Additionally, you can run SWAN on HPC with the SWAN docker image using Apptainer or Singularity.
